# Guia-de-Planejamento-Financeiro
Caderno temático sobre Orçamento e Planejamento Financeiro, criado no NotebookLM. Contém objetivos de estudo, perguntas estratégicas, prompts testados com respostas da IA, resumos estruturados, glossário de termos e prompts reutilizáveis para futuras revisões. Focado em aprendizado ativo e organização do conhecimento financeiro para iniciantes.
